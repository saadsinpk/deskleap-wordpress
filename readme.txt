Add DeskLeap live chat widget and support portal to your WordPress site.

== Description ==

DeskLeap is a modern customer support platform. This plugin adds the DeskLeap chat widget to your WordPress site, allowing visitors to:

* Start live chat conversations with your support team
* Browse knowledge base articles directly from the widget
* Submit and track support tickets

= Features =

* One-click widget installation - no code needed
* Customizable widget color and position
* Configurable greeting message
* Works with any WordPress theme

= Getting Started =

1. Install and activate the plugin
2. Go to Settings > DeskLeap
3. Enter your Organization ID (found in your DeskLeap dashboard under Settings > Widget)
4. Save and your widget is live!

== Installation ==

1. Upload the `deskleap` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings > DeskLeap to configure
